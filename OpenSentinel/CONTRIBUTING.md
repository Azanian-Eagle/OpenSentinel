# Contributing to OpenSentinel

Thank you for your interest in contributing to OpenSentinel! We welcome contributions from everyone.

## Getting Started

1. Fork the repository.
2. Clone your fork: `git clone https://github.com/your-username/OpenSentinel.git`
3. Create a new branch: `git checkout -b feature/my-new-feature`

## Development

- **Backend**: The backend is written in Rust. Please ensure you have the latest stable toolchain installed. Run `cargo test` before submitting.
- **Frontend**: The client is lightweight JavaScript/TypeScript. Keep it under 20kb.

## Code Style

- Follow standard Rust formatting (`cargo fmt`).
- Ensure code is well-documented.

## Pull Requests

- Use a clear and descriptive title.
- Describe the changes in detail.
- Link to any related issues.
- Ensure all tests pass.

## Code of Conduct

Please be respectful and considerate of others. Harassment of any kind will not be tolerated.
